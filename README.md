# Doggo Frontend
## How to run the app
- Install node modules (In the root directory)
```
$ npm install
```

- Run the app (In the root directory)
```
$ npm run start
```

- Paths that are already setup
    -  `/`: The home page
    - `/dog`: The page for dog info
    - `/profile`: The page for user profile
    - `/quiz`: The page for quiz
